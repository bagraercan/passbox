About Passbox

Passbox is a password manager application which allows users to manage their passwords in online and offlines modes.

  • Create your free account
  • Use online or offline or both together as you wish
  • Passwords and usernames (if you have it) are encrypted when you save them
  • Manage all your passwords

Privacy policy

Passbox password manager

The main goal of the application is to store user's data such as passwords, login information and other related data. These data are encrypted before saving. The application contains two modes (online and offline) which users can choose to store their data on an online server or in their own device.

Online

Users can store their data on an online server when they use online mode. Users are registered with their email address and master password. This master password is also encrypted before saving and cannot be seen by any other users or persons. Every user has a uniqe id when when they registered. This id helps them to save their data securelly. The registered accounts (only email) can be seen by developer.

Offline

Users' data are stored on their own devices as encrypted in a secure storage. This data can be seen only by its own user. No data is shared with developer or any other persons.

How is data stored?

Data is stored as encrypted with master password in both modes. This data will be lost permanently when users forgot their master password.

Data/User Privacy

Application does not send, sell, lend or share any kind of user data with anybody. All process begins with user and ends with user.
